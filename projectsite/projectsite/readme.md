# Student Organization Management System

##  Project Description

The Student Organization Management System is a Django-based web application that manages students, organizations, colleges, and academic programs within an institution.

This system helps administrators organize student data, assign students to organizations, and manage institutional structures efficiently using the Django Admin interface.

---

##  Features

- Manage Colleges
- Manage Academic Programs
- Manage Student Records
- Manage Student Organizations
- Assign Students to Organizations
- Track Organization Membership Date
- Search and Filter Records in Admin Panel
- Generate Sample Data using Custom Management Command
- Automatic Timestamp Tracking (created_at, updated_at)
